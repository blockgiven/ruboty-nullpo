# Ruboty::Nullpo

ruboty plugin for ぬるぽ ガッ.

![screenshot](screenshot.png)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-nullpo'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-nullpo
